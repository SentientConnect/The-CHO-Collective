# Happy Monday Message — The CHO Collective

Production-ready Next.js landing page focused on signups for Edwin Edebiri's Happy Monday Message under The CHO Collective.

## Deploy

1. Upload the contents of this folder to the root of the existing GitHub repository.
2. Commit to `main`.
3. Vercel will redeploy the connected project automatically.

## Signup destination

The buttons default to:

`https://www.skool.com/the-cho-collective-8979/about`

To use a different destination, set this Vercel environment variable:

`NEXT_PUBLIC_HMM_SIGNUP_URL`

## Brand assets

The homepage uses one Edwin/CHO Collective hero image. All v0 favicon assets have been replaced with CHO Collective-branded icons.
