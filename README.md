# Happy Monday Message Landing Page

A focused signup landing page for Edwin Edebiri's **Happy Monday Message**, presented by **The CHO Collective**.

## Deploy to Vercel

1. Upload the contents of this folder to the connected GitHub repository.
2. Vercel will automatically build and deploy the Next.js project.
3. The default signup destination is The CHO Collective Skool page.

## Optional custom signup link

To point every signup button to a GoHighLevel form, calendar, or another destination, add this environment variable in Vercel:

```text
NEXT_PUBLIC_HMM_SIGNUP_URL=https://your-signup-link.com
```

Then redeploy the project.
