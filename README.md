# Happy Monday Message — The CHO Collective

Production-ready Next.js landing page for Edwin Edebiri's Happy Monday Message.

## Deploy

1. Upload the contents of this folder to the GitHub repository.
2. Commit the files to `main`.
3. Vercel will redeploy automatically when the repository is connected.

## Signup destination

All calls to action default to:

`https://www.skool.com/the-cho-collective-8979/about`

To change the destination without editing code, add this Vercel environment variable:

`NEXT_PUBLIC_HMM_SIGNUP_URL`

## Media

The homepage hero uses the optimized browser-ready video at:

`public/video/cho-intro.mp4`

The image at `public/video/cho-intro-poster.jpg` is displayed while the video loads.
