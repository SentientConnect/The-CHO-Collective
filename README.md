# The CHO Collective Landing Page

Production-ready Next.js landing page for The CHO Collective.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Vercel Analytics

## Local development

```bash
pnpm install
pnpm dev
```

## Production build

```bash
pnpm build
```

## Vercel deployment

1. Push the contents of this folder to the GitHub repository connected to Vercel.
2. Vercel will detect Next.js automatically.
3. No environment variables are required for the current landing page.
4. Leave Framework Preset as `Next.js`, Build Command as `next build`, and Output Directory blank.

All enrollment calls to action currently lead to:

`https://www.skool.com/the-cho-collective-8979/about`
